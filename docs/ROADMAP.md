# Roadmap

## v0.1 Foundation
Telegram, MEXC public data, scanner, PAPER, OpenAI, Docker.

## v0.2 Paper Portfolio
Virtual balance, PAPER positions, TP1/TP2, breakeven, trailing, history.

## v0.3 Live Smoke Test
Authenticated API preflight, BTC-only 1x live order, attached TP/SL,
double confirmation, position/TP-SL verification, emergency close.

## v0.4 Confirm Trading
Full contract sizing, idempotency, partial fills, reconciliation,
separate TP levels, durable pending orders, daily limits.

## v0.5 Auto Trading
Risk manager, position manager, restart recovery, complete monitoring.
