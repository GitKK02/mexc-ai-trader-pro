# v0.4.0

## Added
- Multi-asset CONFIRM execution.
- Dynamic contract sizing.
- Risk and notional caps.
- Unique externalOid and persistent LIVE journal.
- Per-position emergency close.
- Read-only reconciliation.

## Safety
- AUTO is not implemented.
- Isolated margin is mandatory.
- Every trade requires two confirmations.
- Missing TP/SL triggers an emergency close attempt.
