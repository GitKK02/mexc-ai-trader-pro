# LIVE Smoke Checklist

- [ ] API key has Futures Account Read, Order Details, Order Placing.
- [ ] Withdrawal permission is disabled.
- [ ] API key is bound to the VPS IP when possible.
- [ ] No existing Futures positions.
- [ ] No existing Futures orders.
- [ ] BTC_USDT account mode matches LIVE_POSITION_MODE.
- [ ] LIVE_LEVERAGE=1.
- [ ] LIVE_MAX_NOTIONAL_USDT is acceptable.
- [ ] MEXC app is open on BTC_USDT.
- [ ] User can manually close the position.
- [ ] Telegram confirmation code is private.
